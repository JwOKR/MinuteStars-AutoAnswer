# Changelog

## v4.5.4
- 升级为8方向调整大小（四边+四角手柄，最小 200×150）
- 8个方向均可独立或组合调整面板尺寸

## v4.5.3
- 删除全部拖拽功能（移动面板 + 调节大小），恢复固定位置和尺寸

## v4.5.1
- 优化拖拽功能：添加边界限制、RAF性能优化、触摸支持、视觉反馈

## v4.5.0
- 新增手动调节面板功能：标题栏可拖动移动面板位置，左下角调节手柄可调整面板大小（最小 220×300，最大 600×800）
- 重构拖拽逻辑，标题栏按钮区域不受拖拽影响
- 收起状态下隐藏调节手柄，避免误操作

## v4.4.6
- 修复面板收起/展开不生效的问题：CSS 选择器 `.ata-panel.collapsed` 与 HTML 中的 `#ata-panel` 不匹配
- 新增独立的展开按钮 `#ata-expand-btn`，收起时隐藏收起按钮，展开时显示收起按钮
- 修复倒计时剩余秒数不实时显示：暂停时每秒刷新剩余秒数状态
- 修复倒计时期间无法暂停的根本原因：暂停按钮条件从 `!running` 改为 `!running && !inCountdown`

## v4.4.5
- 修复答题完成后倒计时期间无法暂停的问题：引入 `inCountdown` 标志区分答题阶段和倒计时阶段
- 倒计时阶段暂停按钮保留并变为橙色"⏸ 暂停倒计时"，点击暂停后变为"▶ 继续倒计时"
- 暂停时状态栏显示"⏸ 倒计时已暂停（剩余 Ns）"
- 重置/停止时同步清理所有状态（running/paused/inCountdown/tick）

## v4.4.4
- 新增主面板收起/展开功能：标题栏右侧新增 ▼ 按钮，点击收起主体（仅保留标题栏），再次点击展开

## v4.4.3
- 新增暂停/继续功能：答题过程中可随时点击暂停，循环阻塞等待；继续后从暂停位置恢复答题
- 暂停时控制提交倒计时：暂停倒计时秒数冻结，继续后从剩余秒数继续倒计时
- 面板新增黄色暂停按钮，答题开始时显示，停止/完成/重置时隐藏
- 停止按钮也会同步清理提交倒计时 interval

## v4.4.2
- 修复多选题"没有作答"的问题：根因是 `fill()` 中 `norm(String(answer)).split(',')` 顺序错误，norm() 先吞掉逗号导致 "A,D" 变成 "AD"，无法匹配单个选项；改为先 split 再 norm
- 主面板版本号改为自动读取 `@version` 元数据，不再硬编码

## v4.4.1
- 修复多选题命中但不勾选的问题：checkInput 增加 label 优先触发 + pointerdown/pointerup 事件链 + await 顺序执行防止并行冲突

## v4.4.0
- 主面板 UI 全面升级：暗色主题 + 渐变色 + 统计卡片 + 进度条动画

## v4.3.3
- 重写答案采集逻辑，兼容 MinuteStars 结果页 `.answer-badge.reference` 结构

## v4.3.2
- 修复答案采集误采集用户自己答案的问题

## v4.3.1
- 修复 Tampermonkey 沙箱下 MouseEvent view 报错
