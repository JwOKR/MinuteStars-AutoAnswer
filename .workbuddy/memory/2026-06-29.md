## Gitee 仓库文件云同步 (v4.8.51)
- **背景**：Gitee Gist 只能私有，无法公开分享题库
- **方案**：题库存到 `law-of-order/MinuteStars-AutoAnswer` 仓库文件（tiku.json）
- **关键函数**：
  - `repoRawUrl(filePath)` — 公开 raw 链接（免认证）
  - `repoApiUrl(filePath)` — API 端点（需 Token）
  - `_readRepoFile(filePath)` — 先 raw 后 API 双通道读取
  - `_writeRepoFile(filePath, content, message)` — base64 编码后 PUT
- **分享**：生成时间戳快照 `tiku_share_xxx.json`，返回公开 raw URL
- **导入**：支持粘贴完整 URL（自动识别 http 开头走 fetch）
- `_gistReq` → `_cloudReq`（通用请求函数重命名）
- 配置：`cloudGistId` 保留但不使用，新用 `cloudFilePath`
